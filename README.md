
# 小土的网络工具箱

* Ubuntu 18.04
* g++ 7.4.0
* gcc 7.4.0
* make 4.1

## 依赖:

### 1. gtest
```
$ git clone https://github.com/google/googletest.git
$ cd gooletest
$ mkdir build
$ cd build
$ cmake ..
$ make
$ sudo make install
```
